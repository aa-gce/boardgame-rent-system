# Project Objectives and Target Community

### Overall Goals:

The main goal of this project is to design an online system for renting board games between users; so that people who own board games can register their games in the system and interested people can rent those board games for a certain period of time.
This system allows users to access a variety of games at a lower cost and game owners can also earn money from their unused games.


**Partial Goals:**
- Increase access to board games
- Generate income for game owners
- Better use of unused resources
- Regular management of the rental process
- Increase trust between users
- Reduce the cost of entertainment for users
- Manage damage, delay and deposit
- Promote the culture of group games


### Project Target Community:
The target community of this system includes people who are interested in board games or own board games and want to rent or lease games through an online platform.  
-> Board game enthusiasts  
-> Students and young people  
-> Families  
-> Board game owners  
-> Game bars and entertainment centers  
-> Celebrants and Performers of small gatherings and events.
