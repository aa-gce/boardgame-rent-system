# Project Scenario
Online Board Game Rental System
____________________________________
⚠️ Nowadays, board games have become a popular pastime among families, students, and groups of friends. However, the price of many board games is high, and not everyone can afford to buy a large number of games. On the other hand, many people have games at home that they use less and less after a few times and remain unused.

✅ To solve this problem, an online BOARD GAME RENTRAL SYSTEM is designed. This system provides a platform where people who own board games can register their games in the system and other people can rent those games for a certain period of time.

In this system:
1️⃣ Users first register and log in to their account.
2️⃣ Each user can act as a “game owner” or “rental requester”. The game owner registers his game information in the system; Information such as Name of the game, Game specifications, Physical condition of the game, Rental amount, Deposit, Delivery area, and Game images.
Other users can search for games and register rental requests for a specific period of time.
3️⃣ After submitting the request, the game owner can approve or reject it. If approved, the user pays the rental amount and deposit online, and the status of the request is recorded in the system. After the game is delivered and returned, the owner checks the health status of the game and releases the deposit if there is no damage or delay.
4️⃣ After the game is delivered, the order status is recorded in the system. At the end of the rental period, the user must return the game to the owner. The owner confirms the return status after checking the health and completeness of the game parts. If the game is returned intact and on time, the deposit is released. In case of delay or damage, the system can record a fine or deduction from the deposit.

👨‍💼 The system administrator also monitors user performance, registered games, transactions, complaints, and reports. Also, in the event of a violation, damage to the game, or failure to return on time, the administrator can check the status of the request and take necessary measures.

🚀 This system allows people to experience a variety of games at a lower cost, and game owners can also earn money from their unused games. This system also helps to expand the culture of group games, reduce the cost of purchasing games, and make better use of available resources.
